LLM from Scratch

View on GitHub

Project Overview

The Custom GPT Implementation is a lightweight, from-scratch transformer model built with PyTorch and Hugging Face, providing a full training pipeline and interactive chat interface for flexible text generation without relying on high-level APIs.

Key Features

From-Scratch Transformer Decoder: Implements custom PositionalEncoding, MultiHeadSelfAttention, and FeedForward modules to form a GPT-style decoder stack.

Hardware Optimization: Supports CUDA acceleration, automatic mixed precision (AMP), and PyTorch compilation for efficient GPU training and inference.

Advanced Training Pipeline: Includes cyclical learning rate scheduling (CyclicLR), gradient clipping, interrupt-safe checkpointing, and adaptive learning rate adjustment.

Versatile Text Generation: Offers both standard and streaming token-by-token output modes plus an interactive chat interface with context management.

Checkpoint & Logging: Automated checkpoint saving, resumption, and detailed training logs for robust experiment tracking.

Technical Implementation

Engineered in Python using PyTorch and Hugging Face tokenizers, the model centers on an nn.TransformerDecoder built from custom layers. The training loop leverages torch.cuda.amp for mixed precision, the AdamW optimizer with a CyclicLR scheduler, and tqdm for progress bars. Checkpoint management ensures safe resumption and logging captures loss, LR, and performance metrics.

Unique Approach

Rather than using off-the-shelf transformer wrappers, every component—from embedding and positional encodings to causal masks and token sampling—was implemented manually. Streaming generation and interactive chat are handled at the token level to demonstrate full control over the model’s behavior and output flow.

Learning Outcomes

Gained deep insights into transformer internals, self-attention mechanics, and efficient GPU training techniques. Developed expertise in mixed precision, dynamic learning rates, checkpointing strategies, and building interactive machine learning applications end-to-end.

SPA HTML Local File Explorer

View on GitHub

Project Overview

The SPA HTML Local File Explorer is a browser-based application that allows users to browse, manage, and interact with files stored on their local device without requiring server-side processing.

Key Features

Single-Page Architecture: The entire application runs in a single HTML file, using modern JavaScript to handle all functionality without page refreshes.

File System API Integration: Leverages the browser's File System Access API to securely access the user's local files with permission.

Intuitive Interface: Features a familiar file explorer layout with navigation breadcrumbs, icon/list views, and sortable columns.

File Operations: Supports basic operations like copy, move, rename, delete, and create new folders or files.

Preview Capabilities: Built-in viewers for common file types including text, images, audio, and video.

Technical Implementation

This project was particularly challenging as it required working within browser security constraints while still providing a seamless user experience. I implemented the application using vanilla JavaScript without any frameworks to ensure maximum efficiency and compatibility.

Security Considerations

Security was a primary concern throughout development. The application requests explicit permission from users before accessing any files, never transmits file data over the network, and runs entirely client-side to maintain data privacy.

Learning Outcomes

Building this application deepened my understanding of modern web APIs, browser security models, and efficient client-side application architecture. I learned to create complex functionality that traditionally required desktop applications, all within the browser environment.

Manual Local Music Player

View on GitHub

Project Overview

The Manual Local Music Player is a lightweight desktop application designed to provide users with a simple, customizable way to enjoy their music library without relying on streaming services or internet connectivity.

Key Features

Minimalist Design: Clean, intuitive interface focused on music playback rather than unnecessary features.

Manual Organization: Gives users complete control over how their music is organized, allowing custom playlists, tags, and folder structures.

Audio Customization: Includes equalizer settings, playback speed controls, and volume normalization.

Metadata Editor: Built-in tool for viewing and editing music metadata (artist, album, genre, etc.).

Lightweight Performance: Optimized to use minimal system resources while maintaining high-quality audio playback.

Technical Implementation

This application was built using Electron.js for cross-platform compatibility, with a focus on native file system access for optimal performance. I implemented custom audio processing algorithms to handle various file formats and quality settings.

Unique Approach

Unlike most modern music players that focus on recommendation algorithms and online features, this player intentionally returns to basics, giving users complete control over their listening experience without algorithmic interference.

Learning Outcomes

This project deepened my understanding of audio processing, file system operations, and desktop application development. I gained valuable experience in creating an application that respects user privacy while delivering a focused, high-quality experience.

About "The Three Orbits"

"The Three Orbits" is my philosophical exploration of the systems that shape our world and our minds. At just thirteen years old, I've endeavored to create a comprehensive framework for understanding the interconnected nature of education, technology, governance, and human experience.

Key Themes

Educational Reform: The book critically examines modern education systems that prioritize memorization over understanding and compliance over creativity.

Technological Ethics: I explore artificial intelligence and its impact on society, sharing my personal experience building an LLM and reflecting on the broader implications of technological advancement.

Economic Systems: The text challenges conventional economic structures and proposes alternative models based on value circulation rather than extraction.

Governance: I question the effectiveness of traditional political systems and consider how they might be redesigned to better serve human flourishing.

Personal Journey: Throughout the book, I weave in personal reflections about my relationship with my father, my ambitions, and the limitations I face as a young thinker with boundless aspirations.

Purpose

"The Three Orbits" represents both my analysis of world systems and my personal manifesto. It challenges readers to question established structures and consider alternative possibilities based on principles of justice and human dignity.
